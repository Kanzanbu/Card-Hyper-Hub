About & Help

Everything you need to know about using Card Gaming Hub.

What is Card Gaming Hub?

Card Gaming Hub is a lightweight, beginner-friendly web platform for learning card games. Browse popular games, study interactive strategy tables, and get personalized game recommendations — all without creating an account.

Built for CSC 4370/6370 — Track D: Card Gaming Hub with the Strategy Table Pack and Recommendation Engine UI bonus feature.

How to Use Each Feature

🃏 Games Library

Browse all available card games with difficulty ratings. Click the filter buttons to sort by Easy, Medium, or Hard. Each card links directly to that game's strategy table.

Go to Games

♟ Strategy Table

Use the tabs at the top to switch between games. Each table shows the full turn-by-turn flow with rules and strategy tips. The play area zones illustrate the physical table layout.

Open Strategy

✨ Recommendations

Select your experience level, player count, vibe preferences, and session length. Hit "Get My Recommendations" to see ranked game suggestions with confidence scores and explainability notes.

Get Picks

🎴 Card Flip Demo

On the Home page, hover over the playing cards to flip them. This is a pure CSS interaction — no JavaScript required. Try using your keyboard too (Tab + Enter or Space).

Frequently Asked Questions

Do I need to create an account? +
No account required. Card Gaming Hub is completely open — browse games, read strategies, and use the recommendation engine without signing up.
How does the recommendation engine work? +
The recommendation UI matches your selected preferences (experience, player count, vibe, session length) against each game's attributes. The confidence percentage reflects how many of your preferences align with that game.
Is this site keyboard and screen-reader accessible? +
Yes. All interactive elements have visible focus states, proper ARIA labels, and semantic HTML5 landmarks. Animations respect the prefers-reduced-motion media query. High-contrast text is used throughout.
Does the site work on mobile? +
Yes — the layout is fully responsive using CSS Flexbox and Grid. It's been tested on desktop (1440px), tablet (768px), and mobile (375px) screen widths.
